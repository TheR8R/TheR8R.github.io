If you are using a screen reader either NVDA or JAWS do the following to have access to the simulation controls: If you are using NVDA turn focus mode on, by pressing the NVDA key + space bar. If you are using JAWS untoggle the virtual cursor by pressing the JAWS key + Z and then use the W, A, S and D keys to move around. Also to skip a level use the JAWS key + backspace. If you use W, A, S and D to move, then for the best hearing experience turn off "speak typed character" as well. When you are ready click the play button to start the simulation.
